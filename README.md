# 安全部物语 · Security Dept Story

> 开罗风格的企业安全部门模拟经营游戏（Godot 4.3）— v1.0 完整版

## 玩法

你是一家企业的安全部长。招聘员工、培训技能、升级部门、扩张楼层，抵御源源不断的网络攻击。

### 核心循环
招聘 → 派单处置事件（含分支选择） → 赚钱 → 培训/升级 → 解锁楼层 → 抵御更强攻击 → 冲击年度大奖 → 二周目继承

### v1.0 系统清单
- **AI 像素美术**：6 员工头像（3 种色调变体）/ 6 设施插画 / 4 楼层地板 / 10 事件图标 / 4 张氛围插画（颁奖背景/成就徽章/logo/菜单背景），全部由火山方舟 seedream 生成
- **俯视图办公室**：3 层可扩张楼层（F1 办公区 → F2 运营区 SOC/培训/休息 → F3 研究区 实验室/取证/公关），部门升级解锁楼层；员工像素小人状态机（Idle/Walk/Work/HandleIncident/Rest），头顶气泡显示情绪；点击员工弹详情面板（含技能条/战绩）；SOC 扫描线 + 工位显示器闪烁 + 设施呼吸灯
- **员工养成**：5 维能力 × 6 种专长 × 5 种性格 × 11 种课程（3 个 NG+ 解锁） × 等级系统 × 3 种外貌变体
- **员工转职**：5 级开放 3 条职业路径（红队攻击 / 蓝队防御 / 紫队研究），每条 3 级，被动加成（难度感知/进度/疲劳/经验/RP/培训）
- **员工气泡对话**：员工头顶随机冒出个性化对话（工作/疲劳/培训/紧急/胜利 6 大池 70+ 条）
- **新闻滚动条**：TopBar 下方横向 ticker，循环播报行业动态 + 彩蛋
- **事件选择树**：10 种攻击类型，处置中触发分支决策（蜜罐诱捕/勒索谈判/反制追踪/内鬼处置），选项带技能门槛
- **专班任务**：严重事件 50% 概率标记为专班，需同一员工连续处置，3 天额外 +15RP +30EXP，中途不可换人
- **设施相性 combo**：同层相邻触发 5 种加成（SOC+取证=难度-5%；培训+休息=效果+15%；实验+公关=RP+2/月；SOC+培训=经验+20%；取证+实验=RP+30%）
- **彩蛋事件池**：16 条业界梗随机触发（Log4j 爆发 / 密码贴显示器 / 老板被钓鱼 / 员工直播泄露屏幕 / AI 换脸诈骗 / 开源组件投毒 ...）
- **成就系统**：25+ 成就分 5 类（经营/养成/防御/建设/彩蛋），Toast 弹窗（含徽章图）+ 列表 UI
- **年度奖项**：每年 12 月 4 维评分（防御/培训/响应/研究）vs 5 家虚拟对手公司，颁奖典礼（含舞台背景图）+ 总评第一奖励
- **季度 CEO 评价**：每 3 月 S/A/B/C/D 评级（拦截率+资金+稳定），B 级以上触发预算谈判（稳健/平衡/激进 3 选项）
- **月度行业排名**：每月 1 号刷新，右侧面板实时显示玩家 vs 5 家对手
- **多存档槽**：3 手动 + 1 自动 + 版本号 + NG+ 标记
- **NG+ 二周目**：存活 5 年解锁，选 2 名员工继承（保留 50% 技能） + 解锁高级课程 + 噩梦难度
- **研究点 RP**：实验室产出，用于解锁成就奖励；事件处置也产出

## 操作

- **速度**：⏸ ▶ ▶▶ ▶▶▶
- **办公室 Tab**：俯视员工走动 + 楼层切换 + 点击设施升级
- **事件 Tab**：分配员工处置，注意"需决策"状态触发选择树
- **招聘**：3 候选人选 1（每月刷新）
- **培训**：员工卡片按钮，11 种课程按等级解锁
- **存档**：右上角 💾，支持 3 手动槽 + 1 自动槽
- **成就**：右上角 🏆

## 文件结构

```
project.godot            # autoload 配置
icon.svg
scenes/
  main.tscn              # 总装
  office_view.tscn       # 俯视图
  office_employee.tscn   # 员工 agent
  employee_row.tscn
  incident_row.tscn
scripts/
  core/
    game_state.gd        # 日期/资金/声誉/研究点
    employee_roster.gd   # 员工花名册
    incident_queue.gd    # 事件队列+状态机+选择树
    achievement_mgr.gd   # 成就
    award_mgr.gd         # 年度奖项
    save_mgr.gd          # 多存档槽
    ng_plus_mgr.gd       # 二周目
  models/
    employee.gd          # 含战绩/工位/性格
    incident.gd          # 含状态机+选择树
    incident_choice.gd
    department.gd
  game_data.gd           # 静态配置（部门/课程/事件/成就/选择树）
  game_loop.gd           # 时间驱动
  office_view.gd         # 俯视图容器
  office_employee.gd     # 员工 agent 状态机
  main.gd                # 主界面总装
  employee_row.gd / incident_row.gd
```

存档位置：`user://save_auto.save` / `user://save_slot_{1,2,3}.save`
